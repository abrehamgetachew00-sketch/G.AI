# Personal AI Assistant 🤖

Your own customizable AI assistant that remembers conversations and can be run locally for FREE!

## Features
- 🧠 Long-term memory storage
- 💬 Natural conversation
- 🔒 Privacy-focused (local model option)
- 💾 Persistent memory between sessions
- 🔌 Works with OpenAI or local models (Ollama)

## Quick Start

### Option 1: Local (Free, Private)
```bash
# Install Ollama
curl -fsSL https://ollama.ai/install.sh | sh

# Pull a model
ollama pull llama2

# Run assistant
python personal_ai_assistant.py
